#XM美剧
##介绍
此项目是个人练手用，采用最新技术开发。
>1,接口采用人人美剧API
>
>2,使用retrofit+rxjava+rxandroid+gson访问网络
>
>3,采用5.0风格UI，
>
>4,采用realm作为数据库
>
>5,使用PLDroidPlayer开源播放器
>
>6,图片使用Glide框架
